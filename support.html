Frequently Asked Questions

1. How to use the app?

  • Open an audio file using Command + O
  • The audio will be automatically split into segments
  • Use arrow keys to navigate between segments
  • Type your dictation for each segment
  • Save your work using Command + S

2. Supported File Formats

StepListen supports the following audio formats:

  • MP3
  • WAV
  • M4A

3. Keyboard Shortcuts

  • ⌘O - Open an Audio File
  • Enter - Play/Pause
  • ↓ - Next Segment
  • ↑ - Previous Segment
  • ^R - Toggle Repeat Mode
  • ⌘S - Save Dictation
  • F2 - Split Audio

4. Common Issues

Audio Segmentation: For best results, use audio without background noise. Too much noise may affect the segmentation quality.

Saving Work: Always save your work before closing the app.
